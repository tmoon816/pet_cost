# T-019 Spec: 404 页面 — 通配路由 + 友好提示

- 日期：2026-05-22
- 分类：feature
- 状态：**待审批**

## 一句话

访问不存在的路径时不崩白屏，展示统一风格的 404 页面。

## 方案

### 前端
- `router/index.js` 加通配路由 `/:pathMatch(.*)*`，指向新组件 `NotFound.vue`
- NotFound.vue：居中展示 🐾 404 图标 + 「页面不存在」文案 + «返回首页» el-button（跳 /dashboard）
- 沿用 App.vue 布局（侧边栏+顶栏保持显示，不破坏 chrome）

### 后端
**不动**。

## 验收
- npm run build 通过
- 访问 /abc、/xyz/123 等任意不存在路径显示 404 页面
- «返回首页» 按钮跳 /dashboard
- 侧边栏 / 顶部导航正常显示

## 不做
- 不加 404 埋点统计
- 不改后端
