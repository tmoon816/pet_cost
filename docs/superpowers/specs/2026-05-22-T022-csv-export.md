# T-022 Spec: 数据导出 — 客户 & 账单 CSV 导出

- 日期：2026-05-22
- 分类：feature
- 状态：**待审批**

## 一句话

客户列表页和账单列表页增加「导出 CSV」按钮，下载当前筛选条件下的数据。

## 方案

### 后端
新增两个导出 API：

1. `GET /api/v1/customers/export?q=&sort_by=&sort_dir=`
   - 复用 list_paginated 查询但不分页（全量导出）
   - CSV 列：姓名, 手机号, 备注, 累计消费, 创建时间
   - 返回 `StreamingResponse(media_type="text/csv")`

2. `GET /api/v1/costs/export?pet_id=&customer_id=&category_code=&start_date=&end_date=`
   - 复用 cost crud 查询但不分页
   - CSV 列：日期, 宠物名, 客户名, 服务项目, 金额, 备注
   - 返回 `StreamingResponse(media_type="text/csv")`

CSV 格式要求：
- UTF-8 BOM 前缀（`\ufeff`），Excel 打开不乱码
- 中文表头
- `csv.writer` 标准库，不加第三方依赖

### 前端
- CustomerList.vue 顶部加 «导出 CSV» el-button
- BillList.vue 顶部加 «导出 CSV» el-button
- 点击后调 `exportApi.xxx()`，blob 下载，文件名含日期戳（如 `customers_20260522.csv`）

### 测试
- 后端每个 export 端点加 1 个 pytest 用例（验证 CSV 有 BOM + 表头 + 数据行）

## 验收
- pytest ≥ 95 全过
- npm run build 通过
- 客户列表点击导出 → 下载 customers_YYYYMMDD.csv，Excel 正确显示中文
- 账单列表点击导出 → 下载 costs_YYYYMMDD.csv，Excel 正确显示中文
- 筛选条件影响导出内容

## 不做
- 不加 Excel (xlsx) 格式（CSV 够用）
- 不加定时导出 / 邮件发送
